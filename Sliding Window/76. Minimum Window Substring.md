# 76. Minimum Window Substring

## Problem Statement

Given two strings `s` and `t`, return the minimum window in `s` which will contain all the characters in `t`. If there is no such window in `s` that covers all characters in `t`, return the empty string `""`.

### Example 1:
```
Input: s = "ADOBECODEBANC", t = "ABC"
Output: "BANC"
Explanation: The minimum window substring "BANC" includes 'A', 'B', and 'C' from string t.
```

### Example 2:
```
Input: s = "a", t = "a"
Output: "a"
Explanation: The entire string s is the minimum window.
```

### Example 3:
```
Input: s = "a", t = "aa"
Output: ""
Explanation: Both 'a's from t must be included in the window. Since the largest window of s only has one 'a', return empty string.
```

---

## Approach

To solve this problem, we can use the `Sliding Window` technique. We will maintain a window that expands and contracts as we traverse through the string `s`.

1. We will use two pointers, `left` and `right`, to represent the current window in `s`. Initially, both pointers will be at the start of the string.

2. We will also maintain a frequency count of the characters in `t` and a count of how many characters from `t` are currently in the window.

3. We will expand the `right` pointer to include characters from `s` until we have all the characters from `t` in the window.

4. Once we have a valid window that contains all characters from `t`, we will try to contract the window from the left by moving the `left` pointer to the right, while still maintaining the validity of the window.

5. During this process, we will keep track of the minimum window size and its starting index.


---

## Code Implementation

```cpp
class Solution {
public:
    string minWindow(string s, string t) {
        int n = s.length(), m = t.length();
        int left = 0, right = 0;
        int needed = m, minLen = INT_MAX, startIndex = 0;
        unordered_map<char, int> mpp;
        if(m > n) return "";        
        for(int i = 0; i < m; i++) mpp[t[i]]++;

        while(right < n){
            if(mpp[s[right]] > 0) needed--;
            mpp[s[right]]--;
            
            while(needed == 0){
                if(right - left + 1 < minLen){
                    minLen = right - left + 1;
                    startIndex = left;                    
                }
                mpp[s[left]]++;
                if(mpp[s[left]] > 0) needed++;
                left++;
            }            
            right++;
        }
        if(minLen == INT_MAX) return "";
        return s.substr(startIndex, minLen);
    }
};
```

---

## Complexity Analysis

- **Time Complexity**: O(n + m), where n is the length of string `s` and m is the length of string `t`. We traverse through `s` once and we also need to count the frequency of characters in `t`.

- **Space Complexity**: O(m), where m is the number of unique characters in string `t`. We use a hash map to store the frequency of characters in `t`.
