# 61. Rotate List

## Problem Statement

Given the `head` of a linked list, rotate the list to the right by `k` places.

### Example 1:
```
Input: head = [1,2,3,4,5], k = 2
Output: [4,5,1,2,3]
Explanation:
rotate 1 steps to the right: 5->1->2->3->4
rotate 2 steps to the right: 4->5->1->2->3
```

### Example 2:
```
Input: head = [0,1,2], k = 4
Output: [2,0,1]
Explanation:
rotate 1 steps to the right: 2->0->1
rotate 2 steps to the right: 1->2->0
rotate 3 steps to the right: 0->1->2
rotate 4 steps to the right: 2->0->1
```

---

## Approach

To rotate the linked list to the right by `k` places, we can follow these steps:

1. First, we need to find the `size` of the linked list. We can do this by traversing the list until we reach the end, counting the number of nodes.

2. Once we have the size of the linked list, we can calculate the effective number of rotations needed by taking `k % size`. This is because rotating the list by its size results in the same list.

3. Next, we can make the linked list circular by connecting the last node to the head of the list.

4. After that, we need to find the new tail of the list, which will be the `(size - k - 1)`-th node from the beginning. The new head of the list will be the next node after the new tail.

5. Finally, we can break the circular link by setting the next pointer of the new tail to `null` and return the new head of the list.


---

## Code Implementation

```java
class Solution {
    private ListNode findKthNode(ListNode head, int k){
        ListNode temp = head;
        while(k-- > 0){
            temp = temp.next;            
        }
        return temp;
    }

    public ListNode rotateRight(ListNode head, int k) {
        if(head == null || head.next == null || k == 0) return head;
        int size = 1;
        ListNode temp = head;
        while(temp.next != null){
            temp = temp.next;
            size++;
        }
        k = k % size;
        
        temp.next = head;        
        ListNode kthNode = findKthNode(head, size - k - 1);
        head = kthNode.next;
        kthNode.next = null;
        return head;
    }
}
```

---

## Complexity Analysis

- **Time Complexity**: O(n), where `n` is the number of nodes in the linked list. We traverse the list once to find its size and then again to find the new head after rotation.

- **Space Complexity**: O(1), since we are using only a constant amount of extra space to perform the rotation.
