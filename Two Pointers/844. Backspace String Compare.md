# 844. Backspace String Compare

## Problem Statement

Given two strings `s` and `t`, return `true` if they are equal when both are typed into empty text editors. The character `#` represents a backspace character.

Note that after backspacing an empty text, the text will continue empty.

### Example 1:
```
Input: s = "ab#c", t = "ad#c"
Output: true
Explanation: Both s and t become "ac".
```

### Example 2:
```
Input: s = "ab##", t = "c#d#"
Output: true
Explanation: Both s and t become "".
```

### Example 3:
```
Input: s = "a#c", t = "b"
Output: false
Explanation: s becomes "c" while t becomes "b".
```

---

## Approach

Even though we can solve this problem easily using a `stack`, we can also solve it using the `two pointers` technique in `O(1)` space.

1. We need a helper function that will return the `next valid character position` in the string after considering the backspaces. If the character at the current index is `#`, we will keep moving the pointer back and keep a count of the number of backspaces. If we encounter a valid character, we will check if there are any backspaces left. If there are, we will skip that character and move the pointer back. 

2. We will have `two pointers`, one for each string `s` and `t`. We will keep moving the pointers back until we find a valid character in both strings. If the characters at both pointers are not equal, we will return `false`. 

3. If we reach the beginning of both strings and all characters matched, we will return `true`. If we reach the beginning of one string but not the other, we will return `false`.

4. Continue this process until we have traversed both strings completely.


---

## Code Implementation

```java
class Solution {
    public boolean backspaceCompare(String s, String t) {
        int i = s.length() - 1, j = t.length() - 1;

        while(i >= 0 || j >= 0){
            i = nextValidCharPosition(s, i);
            j = nextValidCharPosition(t, j);

            if(i < 0 && j < 0) return true;
            if(i < 0 || j < 0) return false;
            if(s.charAt(i) != t.charAt(j)) return false;
            i--; j--;
        }
        return true;
    }

    private int nextValidCharPosition(String s, int index){        
        int end = index, backspaces = 0;
        while(end >= 0){
            if(s.charAt(end) == '#'){
                backspaces++;
            }
            else if(backspaces > 0){
                backspaces--;
            }
            else{
                break;
            }
            end--;
        }
        return end;
    }
}
```

---

## Complexity Analysis

- **Time Complexity**: O(n + m), where n is the length of string `s` and m is the length of string `t`. We traverse both strings once.

- **Space Complexity**: O(1), as we are using only a constant amount of extra space.
