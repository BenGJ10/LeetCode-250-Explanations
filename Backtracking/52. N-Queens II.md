# 52. N-Queens II

## Problem Statement
The n-queens puzzle is the problem of placing n queens on an n x n chessboard such that no two queens threaten each other.
Given an integer n, return the number of distinct solutions to the n-queens puzzle.

### Example 1:
```
Input: n = 4
Output: 2

Explanation: There are two distinct solutions to the 4-queens puzzle as shown below.

[[".Q..","...Q","Q...","..Q."],
 ["..Q.","Q...","...Q",".Q.."]]
```

### Example 2:
```
Input: n = 1
Output: 1
```

---

## Approach

How does a `Queen` move in chess? It can move horizontally, vertically, and diagonally. So, when we place a queen on the board, we need to make sure that no other queen is placed in the same row, column, or diagonal.

To solve this problem, we can use a backtracking approach. We will try to place queens `row by row` and check if the placement is valid. 

If at any instance our placement is invalid, we will backtrack and try the next possible placement. If we successfully place queens in all rows, ie, `N-Queens` are placed, we will increment our solution count.

Make use of a helper function `canPlaceQueen` to check if placing a queen at a specific position is valid or not. This function will check the column and both diagonals for any existing queens. 

---


## Code Implementation
```cpp
class Solution {
public:
    int solutions = 0;
    
    bool canPlaceQueen(int row, int col, int n, vector<vector<int>> &board){
        int tempR = row, tempC = col;
        while(tempR >= 0 && tempC >= 0){
            if(board[tempR][tempC] == 1) return false;
            tempR--; tempC--;
        }

        tempR = row, tempC = col;
        while(tempR >= 0 && tempC < n){
            if(board[tempR][tempC] == 1) return false;
            tempR--; tempC++;
        }
        
        tempR = row, tempC = col;
        while(tempR >= 0){
            if(board[tempR][tempC] == 1) return false;
            tempR--;
        }
        return true;
    }

    void backtrack(int row, int col, int n, vector<vector<int>> &board){
        if(row == n){
            solutions++;
            return;
        }

        for(int c = col; c < n; c++){
            if(canPlaceQueen(row, c, n, board)){
                board[row][c] = 1;
                backtrack(row + 1, col, n, board);
                board[row][c] = 0;
            }
        }
    }

    int totalNQueens(int n) {
        vector<vector<int>> board(n, vector<int> (n, 0));
        backtrack(0, 0, n, board);
        return solutions;
    }
};
```

---

## Complexity Analysis

- **Time Complexity**: O(N!), where N is the number of queens. This is because in the worst case, we are trying to place N queens in N rows, and for each row, we have N choices for placing a queen.

- **Space Complexity**: O(N^2) for the board representation.
