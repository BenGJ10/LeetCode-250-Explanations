# 97. Interleaving String

## Problem Statement

Given three strings `s1`, `s2`, and `s3`, determine whether `s3` is formed by an interleaving of `s1` and `s2`.

An interleaving of two strings `s` and `t` is a configuration where they are divided into non-empty substrings such that:

- `s = s1 + s2 + ... + sn`
- `t = t1 + t2 + ... + tm`
- `|n - m| <= 1`

- The interleaving is `s1 + t1 + s2 + t2 + s3 + t3 + ...` or `t1 + s1 + t2 + s2 + t3 + s3 + ...`

### Example 1:
```
Input: s1 = "aabcc", s2 = "dbbca", s3 = "aadbbcbcac"
Output: true
Explanation: One way to obtain s3 is:
- Take "aa" from s1 and "d" from s2, so now we have "aad"
- Take "b" from s1 and "bb" from s2, so now we have "aadbb"
- Take "c" from s1 and "ca" from s2, so now we have "aadbbcbca"
- Take "c" from s1, so now we have "aadbbcbcac"
``` 

### Example 2:
```
Input: s1 = "aabcc", s2 = "dbbca", s3 = "aadbbbaccc"
Output: false
Explanation: Notice how it is impossible to interleave s2 with s1 to get s3.
```

### Example 3:
```
Input: s1 = "", s2 = "", s3 = ""
Output: true
``` 

---

## Approach

Lets define a recursive function `matchChars(i, j, k)` that checks if the first `k` characters of `s3` can be formed by interleaving the first `i` characters of `s1` and the first `j` characters of `s2`.

If at any index `k` in `s3`, the character matches with the character at index `i-1` in `s1`, or the character at index `j-1` in `s2`, we can recursively check for the remaining characters by moving the respective pointers back.

If `k` becomes 0, it means we have successfully matched all characters of `s3` with interleaving of `s1` and `s2`, so we return true. If we exhaust all characters in `s1` and `s2` without matching all characters in `s3`, we return false.

---

## Code Implementation

```cpp
class Solution {
public:
    vector<vector<int>> dp;
    
    bool matchChars(int i, int j, int k, string &s1, string &s2, string &s3){
        if(k == 0) return i == 0 && j == 0;
        if(dp[i][j] != -1) return dp[i][j];
        
        if(i > 0 && s1[i - 1] == s3[k - 1]){
            if(matchChars(i - 1, j, k - 1, s1, s2, s3)){
                return dp[i][j] = true;
            }
        }
        if(j > 0 && s2[j - 1] == s3[k - 1]){
            if(matchChars(i, j - 1, k - 1, s1, s2, s3)){
                return dp[i][j] = true;
            }
        }
        return dp[i][j] = false;
    }

    bool isInterleave(string s1, string s2, string s3) {
        int n = s1.length(), m = s2.length(), k = s3.length();
        this->dp.assign(n + 1, vector<int> (m + 1, -1));
        return matchChars(n, m, k, s1, s2, s3);
    }
};
```

---

## Complexity Analysis

- **Time Complexity**: O(n * m), where n and m are the lengths of `s1` and `s2` respectively. This is because we are using a 2D DP table of size (n+1) x (m+1) to store results for subproblems.

- **Space Complexity**: O(n * m) for the DP table. In addition, the recursion stack can go up to O(n + m) in the worst case, but since we are using memoization, it will be limited to O(n * m) as well.
