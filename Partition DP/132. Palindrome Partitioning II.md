# 132. Palindrome Partitioning II

## Problem Statement

Given a string `s`, partition `s` such that every substring of the partition is a palindrome. Return the minimum cuts needed for a palindrome partitioning of `s`.

### Example 1:

```
Input: s = "aab"
Output: 1
Explanation: The palindrome partitioning ["aa","b"] could be produced using 1 cut.
```

### Example 2:

```
Input: s = "a"
Output: 0
```

### Example 3:

```
Input: s = "ab"
Output: 1
```

---

## Approach

The question demands us to find the `minimum cuts` needed to partition a string such that every substring is a palindrome.

**Where can we make the cuts?** We can make the cuts at every index of the string. For example, for the string "aab", we can make cuts at index 0, 1, and 2.

1. We can use a recursive function to explore all possible cuts in the string. Go through the string and check if the substring from the current index to the next index is a `palindrome`.

2. If it is a palindrome, we can make a `cut` and recursively call the function for the remaining substring.

3. We can use a `dp` array to store the minimum cuts needed for each index of the string. If we have already computed the minimum cuts for a particular index, we can return that value instead of recomputing it.

4. Finally, we return the minimum cuts needed for the entire string.

---

## Code Implementation

```java
class Solution {
    int[] dp;
    
    private boolean isPalindrome(int l, int r, String s){
        while(l <= r){
            if(s.charAt(l) != s.charAt(r)) return false;
            l++; r--;
        }
        return true;
    }
    
    private int partition(int i, String s){
        if(i == s.length()) return 0;
        if(dp[i] != -1) return dp[i];

        int minCost = Integer.MAX_VALUE;
        for(int j = i; j < s.length(); j++){
            if(isPalindrome(i, j, s)){
                int cost = 1 + partition(j + 1, s);
                minCost = Math.min(minCost, cost);
            }
        }
        return dp[i] = minCost;
    }
    
    public int minCut(String s) {
        int n = s.length();
        this.dp = new int[n];
        Arrays.fill(dp, -1);

        return partition(0, s) - 1;
    }
}
```

---

## Complexity Analysis

- **Time Complexity**: O(n^2), where n is the length of the string. The recursive function explores all possible partitions, and for each partition, it checks if the substring is a palindrome, which takes O(n) time in the worst case.

- **Space Complexity**: O(n) for the dp array and O(n) for the recursion stack, leading to a total of O(n) space complexity.
