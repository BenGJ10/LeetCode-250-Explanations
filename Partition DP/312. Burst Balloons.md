# 312. Burst Balloons

You are given `n` balloons, indexed from 0 to n - 1. Each balloon has a number on it represented by an array `nums`. You are asked to burst all the balloons. If you burst the `i`th balloon, you will get `nums[i - 1] * nums[i] * nums[i + 1]` coins. If `i - 1` or `i + 1` goes out of bounds of the array, then treat it as if there is a balloon with a 1 painted on it.

Return the maximum coins you can collect by bursting the balloons wisely.

### Example 1:

```
Input: nums = [3,1,5,8]
Output: 167
Explanation:
nums = [3,1,5,8] --> [3,5,8]
coins = 3 * 1 * 5 = 15
nums = [3,5,8] --> [3,8]
coins = 3 * 5 * 8 = 120
nums = [3,8] --> [8]
coins = 1 * 3 * 8 = 24
Total coins = 15 + 120 + 24 = 167
```

### Example 2:

```
Input: nums = [1,5]
Output: 10
```

--- 

## Approach

We can solve this problem using a `Partition DP` approach. The idea is to consider the last balloon to burst in a given range of balloons.

If we use the `front partition` approach, this problem becomes a bit tricky because the coins we get from bursting a balloon depend on the adjacent balloons, which may have already been burst.

So we have to consider the `last balloon` to burst in a given range. If we know the last balloon to burst, we can calculate the coins we get from it and then recursively solve for the left and right subarrays.

1. We can use a recursive function to explore all possible last balloons to burst in the given range of balloons.

2. For each last balloon to burst, we can calculate the coins we get from it and then recursively call the function for the left and right subarrays.

3. `Coins = nums[l - 1] * nums[last] * nums[r + 1] + leftHalf + rightHalf`

4. We can use a `dp` array to store the maximum coins we can collect for each range of balloons. If we have already computed the maximum coins for a particular range, we can return that value instead of recomputing it.

5. Finally, we return the maximum coins we can collect for the entire range of balloons.

## Code Implementation

```java
class Solution {
    int[][] dp;

    private int partitionDP(int l, int r, List<Integer> nums){
        if(l > r) return 0;
        if(dp[l][r] != -1) return dp[l][r];
        int maxCostSoFar = Integer.MIN_VALUE;

        for(int idx = l; idx <= r; idx++){
            int currCost = nums.get(idx) * nums.get(l - 1) * nums.get(r + 1);
            int leftHalf = partitionDP(l, idx - 1, nums);
            int rightHalf = partitionDP(idx + 1, r, nums);
            maxCostSoFar = Math.max(maxCostSoFar, currCost + leftHalf + 
            rightHalf);
        }
        return dp[l][r] = maxCostSoFar;
    }

    public int maxCoins(int[] input) {
        int n = input.length;
        
        List<Integer> nums = new ArrayList<>();
        nums.add(1);
        for(int i = 0; i < n; i++) nums.add(input[i]);
        nums.add(1);

        this.dp = new int[n + 2][n + 2];
        for(int i = 0; i < n + 2; i++) Arrays.fill(dp[i], -1);
        return partitionDP(1, n, nums);
    }
}
```

---

## Complexity Analysis

- **Time Complexity**: O(n^3) - We are using a nested loop to calculate the maximum coins for each subarray, and we have n^2 subarrays to consider.

- **Space Complexity**: O(n^2) - We are using a 2D array to store the results of subproblems, which takes up O(n^2) space.
